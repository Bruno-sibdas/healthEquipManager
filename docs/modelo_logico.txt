Categoria

descricao


2. Localizacao

Localizacao
-----------

3. Equipamento

Equipamento
------------

UQ codigo_interno

FK id_localizacao

modelo
numero_serie
fabricante

ano_fabrico
custo_aquisicao

4. Fornecedor

Fornecedor
----------
PK id_fornecedor

pessoa_contacto

5. EquipamentoFornecedor

EquipamentoFornecedor
---------------------
PK id_equipamento, id_fornecedor
FK id_equipamento
FK id_fornecedor
tipo_relacao


6. Documento

Documento
---------

FK id_equipamento
FK id_fornecedor
tipo_documento

data_documento
data_validade

caminho_ficheiro


7. Utilizador

Utilizador
----------

UQ email
nome
email
password
perfil
utilizador_ativo


RELACIONAMENTOS

Categoria 1:N Equipamento
Uma categoria pode estar associada a vários equipamentos.
Cada equipamento pertence a uma única categoria.

Localizacao 1:N Equipamento
Uma localização pode conter vários equipamentos.
Cada equipamento encontra-se numa única localização atual.

Equipamento 1:N Documento
Um equipamento pode ter vários documentos associados.
Cada documento pertence a um equipamento.

Fornecedor 1:N Documento
Um fornecedor pode estar associado a vários documentos.
Cada documento pode estar associado a um fornecedor.

Equipamento N:M Fornecedor
Um equipamento pode ter vários fornecedores.
Um fornecedor pode estar associado a vários equipamentos.
Esta relação é resolvida pela tabela EquipamentoFornecedor.
